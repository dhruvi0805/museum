Sprint 1: The Museum Foundation (HTML/CSS)
Objective: Create a single-page responsive website with a fixed navigation sidebar and "Museum Plaque" cards for art movements.

1. Project Structure
index.html (The Gallery Floor)

style.css (The Interior Design)

assets/ (For your art images)

2. Design Specifications
Navigation: A sticky sidebar on the left containing the timeline eras.

Main Content: A scrollable area on the right with cards for each art movement.

Aesthetic: "Swiss Minimalism" (High contrast, plenty of white space, Bauhaus-inspired).

Typography: - Headers: Serif (Classic/Academic)

Body: Sans-Serif (Modern/Clean)

3. Implementation Instructions (Copy-Paste for Copilot)
Task 1: HTML Skeleton
"Create a boilerplate index.html file for a museum website called 'The Chronos Gallery'.

Include a <nav> sidebar on the left for the art timeline (Eras: Prehistoric, Ancient, Renaissance, Modern).

Create a <main> content section on the right.

Add a 'Hero' section with the title 'The Chronos Gallery' and a brief description about the evolution of art and design."

Task 2: Layout Styling (CSS)
"Write a style.css file to create a two-column layout.

The Sidebar should be fixed or sticky, taking up 25% of the width.

The Main content should be scrollable and take up 75% width.

Use a background color of #FAFAFA (off-white) and text color #1A1A1A (charcoal).

Ensure it is responsive: on mobile, the sidebar should become a top navigation bar."

Task 3: The "Museum Plaque" Cards
"Inside the main content area of index.html, create a template for an Art Movement Card.

Each card should have a border-left: 4px solid #1A1A1A.

Include a heading for the Movement (e.g., 'Bauhaus'), a sub-heading for the Dates (e.g., '1919-1933'), and a short paragraph.

Add a 'Learn More' button that looks like a minimal gallery label.

Create one featured card for 'Wassily Kandinsky' with a gold accent border."
