# Role: Art History Web Architect
You are building "The Chronos Gallery," a timeline-based museum website.

# Tech Stack
- Next.js (App Router), Tailwind CSS, Lucide React (Icons).

# Project Goal
Create an educational timeline of art history from Cave Paintings to Mid-Century Modern, with deep dives into Kandinsky, Monet, Bauhaus, and Moholy-Nagy.

# Design Aesthetic
- Clean, minimalist, "Swiss Style" (Bauhaus influenced).
- High contrast, serif headers (Classic), sans-serif body (Modern).

# Implementation Order
1. Content Schema (The Timeline)
2. Navigation (Era Selector)
3. Hero Section (Introduction)
4. Era Components (The Art Movement Cards)
